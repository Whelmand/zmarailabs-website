# Zmara iLabs Website

Statische website voor Zmara iLabs, geschikt voor hosting via GitHub Pages op `zmarailabs.nl`.

## Bestanden

- `index.html` - homepage met secties voor over, projecten, privacy/support en contact.
- `privacy.html` - privacybeleid-template voor apps en tools van Zmara iLabs.
- `styles.css` - responsive dark-tech styling met CSS variables.
- `script.js` - mobile menu, smooth scrolling en huidig jaar in de footer.

## Lokaal bekijken

Open `index.html` direct in een browser. Er is geen backend of build-stap nodig.

## GitHub Pages instellen

1. Open de repository op GitHub en ga naar **Settings**.
2. Ga naar **Pages**.
3. Kies bij **Source** voor **Deploy from a branch**.
4. Selecteer de **main** branch en de **root** folder.
5. Stel bij **Custom domain** het domein `zmarailabs.nl` in.

Controleer daarna of de DNS-records voor `zmarailabs.nl` naar GitHub Pages verwijzen.
